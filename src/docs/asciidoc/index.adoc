== 개요
이 문서는 테스트 기반으로 생성된 스니펫을 포함합니다. 실제 테스트 실행 후 생성된 스니펫을 include 합니다.

== 사용자 API 예시
ifdef::snippets[]

include::apis/user-create.adoc[]
include::apis/user-list.adoc[]
include::apis/user-get.adoc[]
include::apis/user-update.adoc[]
include::apis/user-delete.adoc[]

endif::[]

== 오류 응답 포맷(예시)
[cols="1,1,3", options="header"]
|===
|필드|타입|설명
|code|String|에러 코드
|message|String|사람이 읽을 수 있는 메시지
|traceId|String|추적을 위한 식별자(선택)
|===

== 참고
- 스니펫은 `build/generated-snippets` 경로에 생성됩니다.
- HTML 생성: `gradlew.bat asciidoctor` 실행 후 `build/docs/asciidoc/index.html` 확인.
